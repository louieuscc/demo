# 露营主要设备



**一、天幕/车边幕**

> 车边幕与天幕基本相同，车边幕一面长边可固定在车顶，另一长边固定于地面。

1. 天幕一张

2. 天幕杆2支

3. 天幕杆固定器2个（无需他人协助即可搭杆，并保证杆体垂直）

4. 防风绳6-8根（另购双眼绳扣6-8枚）

5. 地钉6-8支（20cm款）

6. 硬塑料榔头一把（建议颜色鲜艳款，易找）

   

*注1：防风绳也可购买带拉紧器的套装，搭建更方便，无需双眼绳扣*

*注2：车边幕需加购车顶吸盘2-3个，另一边用地钉将防风绳固定于地面*

*注3：有些露营基地提供木基座，帐幕可搭建其上，但需另购鱼骨钉而非普通地钉以固定风绳*



**二、帐篷**

> 建议选择伞式自动收放帐篷，无需额外购买风绳、龙骨杆和地钉。收放自如，方便快捷。

1. 自动收放帐篷一顶

2. 地面防潮布

3. 气垫床

4. 充、放两用气泵（用于气垫床）

5. 睡袋

6. 露营灯

7. 电动吹气筒（用于清扫）

   

**三、露营行李车**

> 如果营地距离停车处较远，装备需放入行李车自行运输

1. 建议购买轮子稍大的款型，更稳，不颠簸
2. 另购相应尺寸的蛋卷桌卡在车顶，秒变为餐桌



**四、饮食设备**

> 电车可用放电枪为电子烹饪设备供电，油车可自备卡式炉和GAS气罐。

1. 卡式炉/放电枪
2. 折叠式锅、壶套装（用于卡式炉）
3. 电炉、电水壶、电汤锅（用于电车）
4. 电源延长插排
5. 桶装纯净水抽水阀
