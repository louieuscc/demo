And try some features on Sourcetree.

Try commit a file on different host, then check what happened on local Sourcetree:

1. 本地文件改动并保存（未add或commit）, Sourcetree能感应并提示后续操作
2. 在本地sourcetree输入注释并`commit`
3. 继续操作，`push`之前改动
4. Push成功，本地与远程同步
5. 故本地Sourcetree能感知外地分机对共有文件的改动，也无法感知远程库对应文件的变化
